# 3 运算定律

## 单元目标

1.使学生认识加法交换律、结合律，乘法交换律、结合律和分配律，理解加法和乘法运算定律的内涵，并能运用运算定律进行一些简便计算。

2.使学生经历运算定律的归纳与概括过程，体验数学模型的建构与解构过程，积累数学基本活动经验。

3.使学生能够结合具体情况，灵活选择合理的算法，在解决问题的过程中，初步感受数学与现实生活的联系，培养学生用所学知识解决简单的实际问题的能力。
